# AlphaStack v0.2 Architecture

## Core principle

Separate the system into:
- alpha-core: reusable research engine
- skill layer: Claude / Codex front-end orchestration
- adapter layer: connection to your real framework

## Major modules

- Generator
- Formalizer
- Validator
- Deduper
- Ranker
- Memory
- Evolver
- Exporter
