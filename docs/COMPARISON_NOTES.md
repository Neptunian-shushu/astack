# Comparison Notes

AStack v0.2 is inspired by, but not identical to, recent alpha-mining research.

## Similar to CogAlpha
- iterative generate / validate / evolve loop
- emphasis on structured search over naive formula stacking
- quality control before admission

## Similar to FactorMiner
- modular skill architecture
- memory-guided exploration
- library-aware design direction
- framework-agnostic philosophy

## Different from both
- explicitly designed as a reusable core + adapter + skill package
- meant to be embedded into your own framework or another team's framework
- emphasizes software architecture and long-term maintainability as much as search logic
