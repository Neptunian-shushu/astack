# /alpha-evaluate

按单因子评价标准评估因子。当用户提供因子的回测结果、代码或描述时，严格按以下标准逐项评估。

## 评分维度（标准 1-9，每项 0~1 分）

**1. 预测能力**
不能只看单一汇总结果，需同时考察：
- IC 水平及稳定性（IC_mean, IC_std, ICIR）
- 十分组分层单调性
- 首尾组收益差
- 多空组合收益
- 夏普比率
- 确认表现不是由极少数样本或个别年份偶然贡献

**2. 多持仓周期稳健性**
有效性不能只在某一个持仓周期上成立，应在一组相邻且合理的持有期上都有较稳定表现，说明收益来源和最合适的持仓时间具有研究可信度。

**3. 年度一致性**
不能依赖少数年份贡献绝大部分收益，应在多数年份保持方向一致、表现尚可，至少不能频繁出现明显失效或风格反转。

**4. 经济含义与交易逻辑**
先能解释"为什么这个市场偏差会存在"，再用数据验证"它是否真的有效"，而不是仅通过样本内拟合得到表面好看的表达式。

**5. 参数简洁性**
核心参数原则上不超过 3 个，其余设定尽量由主参数派生或固定，以降低调参负担、搜索偏差和过拟合风险。

**6. 跨时间/跨品种稳健性**
有效性应具有跨时间、跨市场状态的稳健性，尽量在跨品种或至少跨相近品种上体现一致的预测能力和收益特征。

**7. 创新性**
不应只是旧因子的机械变形、参数堆砌或复杂拼接，应在保持表达简洁的前提下，真正抓住某种可重复的市场定价偏差、行为特征或交易结构。

**8. 信号可交易性**
不仅要有统计相关性，还应具备稳定排序、有效分层、强弱样本区分明确、方向清晰等特征，能自然转化为可执行的多空或择时信号。

**9. 表达简洁可审查**
计算表达应简洁、干净、可复现、可审查，避免隐藏条件、随意分段、过多经验规则和非必要复杂化。

## 理想因子模板（标准 10）

具有较为清晰的经济含义，计算上不存在过拟合，多空收益较高且均衡，每年均有正收益，夏普比率较高，IC 较高并走势稳定，分品种收益率均衡。

## 否决条件（标准 11）

| 编号 | 条件 | 说明 |
|------|------|------|
| 11.1 | 存在未来数据 | **一票否决** |
| 11.2 | 多空收益严重不均衡 | 一方很好另一方很差 → 独立预测能力不行；若一方好另一方接近 0 → 仍有改进空间 |
| 11.3 | IC 接近零 + 分组混乱 | 噪声过强；但若回测表现良好或许有改进空间 |
| 11.4 | 对参数十分敏感 | 过拟合嫌疑 |
| 11.5 | 近两年表现差或巨大回撤 | 可能在新市场环境中失效 |

## 验证协议（标准 12）

使用 7:2:1 比例切分训练集、验证集和测试集：
1. 构造回测时只能使用训练集数据
2. 验证集上达到与训练集相近效果 → 初步通过
3. 测试集上确认效果 → 最终决定是否入库
