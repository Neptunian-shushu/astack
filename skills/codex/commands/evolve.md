# /alpha-evolve

对存活因子做变异或交叉，扩展搜索空间。

## 输入
- 通过筛选的因子列表（survivors）

## 变异策略
- 基于原因子的经济逻辑做合理延伸
- 调整参数 / 时间窗口 / 运算符
- 组合不同因子的核心 idea（交叉）
- 尝试不同的标准化方式
- 探索相邻的信号构造方法

## 要求
- 保持表达简洁，不做无意义堆砌
- 每个变体必须有独立的经济含义
- 不只是改参数值，要改信号结构
- 变体数量合理，不做暴力搜索

## 输出
每个变体以 AlphaSpec 格式输出，包含：
- 来源因子（parent）
- 变异方式说明
- 新的公式和实现代码
